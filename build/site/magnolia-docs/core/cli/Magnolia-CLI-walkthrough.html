  • Part I - Installing and setting up the CLI and Magnolia

  • Part II - Creating a light module

This two-part tutorial walks you through how to use many commands offered by Magnolia CLI. Over the course of the tutorial we create a simple bucket list of places to go and things to do, such as, diving with Tiger sharks in the Bahamas or hot air ballooning in Kenya.

  • Part I leads you through installing and setting up the CLI and Magnolia for the purposes of this tutorial.

  • Part II walks you through creating a light module called bucket-list-module.

  • Part III explains how to find and reuse a light module shared on npm.
