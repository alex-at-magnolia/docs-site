ecomfn

The ecomfn templating functions provide access to the products from your external e-commerce solution in your FreeMarker templates.

The ecomfn templating functions are provided by the E-commerce templating module. The Java class that implements the methods is EcommerceTemplatingFunctions.Related topics:

  • About templating functions
